# Top Menu & View Options

Top Menu

- Search: opens the global search modal (`Ctrl+Shift+F`) to find items, docs or UI pages.
- Views / Scenarios: use the View menu to load saved views or manage scenarios. Non-default views filter the Sidebar lists when active.
- Tools: quick access to utilities such as Export, Import or plugin management when available.
- Help: opens the Help modal which lists the documentation pages (from `www/docs/`).

View Options (summary)

- Timeline Scale: available in both the Top Menu and Sidebar; choose the time scale appropriate for your planning horizon.
- Condensed / Compact Cards: hide less important fields to show more cards at once.
- Dependencies: toggle display of dependency links between cards.
- Unplanned / Unassigned toggles: filter tasks that lack dates or team allocations.

Using Views

- Create and save a View to capture a particular combination of selected projects, teams, filters and expansion toggles.
- Loading a saved View applies its selections to the Sidebar and updates the timeline immediately.
