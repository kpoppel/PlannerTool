# Advanced Filters — Practical Examples

These examples show how to combine Sidebar filters to answer common questions.

Example 1 — Show only unplanned and unallocated Features
1. Under Schedule, enable `Unplanned` and disable `Planned`.
2. Under Allocation, enable `Unallocated` and disable `Allocated`.
3. Optionally restrict Task Types to `Feature`.

Example 2 — Inspect a single team's active Features
1. Under Allocations, select the team you want to inspect.
2. Under State, pick `Active` (or the relevant state).
3. Use Timeline Scale to focus on the delivery window.

Example 3 — Find items with missing parents
1. Under Hierarchy, enable `No Parent`.
2. Combine with Project filters to limit scope.

Example 4 — Find all items hierarchically linked for a project
1. Select the plan for the project
2. Under Expand, click the `Parent/Child Link`
3. Under Hierarchy, select `Linked` and `Parent`.
4. Use Timeline Scale to focus on the delivery window.

Tip: use the Data Funnel counters to confirm the dataset size before performing bulk edits.
